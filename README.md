# axolartl

## Running Commands

Run database:
Download mysql
Run the axolartl_index/create_database.sql scipt

Run backend:
```
cd axolartl/query_environment/qe_app/qe_backend
node server.js
```

Run frontend:
```
cd axolartl/query_environment/qe_app
npx expo start
```

Run tests:
```
cd axolartl/query_environment/qe_app
npm test
```

Use the Expo Go app to preview on a mobile device by scanning the QR code.
